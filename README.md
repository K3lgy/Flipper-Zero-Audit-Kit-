# Flipper-Zero-Audit-Kit-
Kit d'audit de sécurité physique pour Flipper Zero. Parse automatiquement les exports de la carte SD (RFID/NFC, Sub-GHz, WiFi, Bad USB) et génère un rapport PDF professionnel avec scoring de criticité, vecteurs d'attaque et recommandations priorisées. Idéal pour les missions de pentest physique.
